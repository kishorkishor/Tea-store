# ChaiBari - Premium Tea E-commerce

A modern e-commerce platform for premium tea built with Next.js, TypeScript, and Supabase.

## Environment Variables

Create a `.env.local` file with:

```
NEXT_PUBLIC_SUPABASE_URL=your_supabase_url
NEXT_PUBLIC_SUPABASE_ANON_KEY=your_anon_key
```

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm start
```

## Deployment

This project is configured for Netlify deployment. Set environment variables in Netlify dashboard.
